大益爱心基金会
  大益爱心基金会以弘扬“惜茶爱人”的茶道精神，扶贫帮困、助学育才、奉献爱心、关爱社会，构建有影响力的校园公益品牌为宗旨，由云南勐海茶厂和云南大益茶业集团共同发起，成立于2007年12月12日。2013年被评为5A级基金会。
青年益工社
  2015年底，大益爱心基金会与中国青少年发展基金会、中国青年网共同发起“青年益工社”项目，致力于打造有效衔接在校大学生成长发展需求与社会公益资源的枢纽，以价值传递为理念，以能力提升为抓手，通过资金、培训等资源支持，推动大学生自主开展公益、文化、创业活动，传播“惜茶爱人”的公益理念，促进大学生形成每年至少参与20个小时公益服务的公益习惯，帮助大学生成长为具有责任感、使命感的新一代青年。
益公益
  扶持培育高校公益社团发展，培养大学生公益服务习惯，传播“惜茶爱人”的公益理念。 资金支持：活动经费资助；场地支持：青年茶庭&茶室免费使用；能力建设：公益骨干培训、NGO导师指导；服务对接：对接优质社会组织公益项目。
益创业
  打造资源共享的开放平台集群，以龙头企业带动大学生创业的模式，切实锻炼大学生创业能力、提高大学生群体性创业成功率。 专业创业培训：整合顶级培训资源，开展创业相关培训；创业督导：筛选优质项目，配备企业家导师督导；创业项目孵化：结合本地政策，为创业团队路演、获取投资提供机会。
益文化
  扶持促进高校开展传统文化活动，传承与传播中华优秀传统文化，引导大学生君子品格与人文素质养成、塑造正向价值观。 梧桐茶会：扶持有人文实力的大学生团队开展梧桐茶会；茶道文化：通过茶道课程、青年茶道师、茶文化进校园等项目，打造大学生茶文化交流圈；传统文化：培育传统文化类社团发展。
