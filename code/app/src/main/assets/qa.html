找一群有社会责任的公益伙伴
找一支撬动资源的精锐团队
找一批面向未来的领袖人才
找一个敢想、敢做、敢担当的你

Q2：我要怎么组建青年益工社？
所在地区青少年发展基金会或所在院校与大益爱心基金会签订协议成立在校合法登记的社团.社团项目属于公益、创业、文化范畴 认可大益爱心基金会公益理念，对青年益工社有热情、有期盼

Q3：我要如何加入青年益工社？
登录青年益工社APP,找到所在院校青年益工社，申请成为社员,所在院校尚未建立青年益工社，恭喜你，可以申请组建青年益工社啦

Q4：我能在青年益工社得到哪些支持？
每个青年益工社，每年根据申报项目，可获得不超过5000元的经费支持
成为青年益工社骨干，将获得免费参加骨干能力提升培训的资格
开展青年益工社项目，将有机会登上全国大赛的舞台
参与青年益工社服务，累积公益服务时间，获得公益服务认证
更多意想不到的福利，加入青年益工社，让年轻的梦想飞向远方
